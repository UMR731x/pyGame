# 🎮 IdleTextRPG - 纯原生 Python 自动化挂机文字游戏

这是一个专为“摸鱼”或后台测试设计的文字冒险（放置类）游戏脚本。它完全不需要用户进行任何键盘或鼠标交互，启动后即可自动在后台打怪、升级、攒金币、购买神装。

## ✨ 游戏特色

- **零交互全自动**：主角会自动前往迷宫冒险、遭遇怪物、计算战斗、回城回血以及购买更强的装备。
- **轻量且低消耗**：代码仅 100 多行，战斗与休息之间有休眠机制，不会占用过高 CPU。
- **支持后台静默**：提供无窗口后台运行方案，随时随地静默挂机。

## 🚀 快速开始

### 1. 正常运行（带控制台输出）


> 注意设置运行环境编码为utf-8，否则会导致文字显示不正常！

直接在终端或命令行中运行脚本，可以实时看到主角的冒险日志：

```bash
python game.py
```

### 2. 后台静默挂机（不打扰工作）

#### 🪟 Windows 系统
1. 将游戏脚本的后缀名从 `.py` 改为 **`.pyw`**（例如 `game.pyw`）。
2. 双击运行 `game.pyw`。游戏将在系统后台完全隐形运行，不会弹出任何黑色命令行窗口。
3. *提示：若想停止游戏，需要在 Windows 任务管理器中找到 `pythonw.exe` 进程并结束它。*

#### 🐧 Linux / macOS 系统
使用 `nohup` 命令让脚本在后台持续运行，并将游戏日志实时写入本地文件：

```bash
nohup python3 game.py > game.log 2>&1 &
```
*   **查看挂机进度**：输入 `tail -f game.log`
*   **停止游戏**：通过 `ps -ef | grep game.py` 找到进程号，然后使用 `kill -9 进程号` 结束。

## ⚔️ 游戏核心数值机制

*   **成长体系**：击杀怪物可获得经验值（EXP）和金币（Gold）。经验满足时自动升级，升级后生命上限（Max HP）和攻击力（Attack）按百分比大幅提升。
*   **动态难度**：主角只会遭遇与自身等级相匹配的迷宫怪物。随着等级提升，会逐步解锁“哥布林百夫长”、“深渊骨龙”等强力 Boss。
*   **智能回城**：当主角生命值低于 30% 或触发随机事件时，会自动返回城镇旅馆休息。回城会瞬间补满生命值，并自动消费金币购买商店里更强力的武器。
*   **隐藏奇遇**：每次击杀怪物有 5% 的极低概率触发隐藏奇遇，意外捡到【传世护符】，永久额外叠加攻击力。
*   **死亡惩罚**：若不慎被怪物击败，主角会丢失 20% 的现金，并保留 10% 的生命值惊险逃回城镇。

## 🛠️ 代码结构

- `player`: 存放主角当前的等级、血量、金币及装备数据。
- `monsters_pool` & `weapons_pool`: 怪物数值库与商店装备库。
- `log()`: 格式化输出函数，带时间戳，并强制刷新标准输出缓冲区，确保后台挂机时日志能实时写入文件。
- `main_adventure()`: 核心游戏主循环，控制着冒险、战斗与休息的节奏。

## 🛑 安全退出

在常规控制台运行模式下，随时按下 **`Ctrl + C`** 即可停止游戏。游戏在退出时会触发结算，打印出主角最终的挂机成果（等级、装备和总资产）。
